## 😎 맡은 업무

* 2차원 배열을 이용하여 장기판의 위치 구현
* Drag & Drop으로 기물들을 움직임
* 각 기물들의 이동로직 구현(DFS, 백트래킹 알고리즘)

&nbsp;

## ❕ 깨달은 점

* 2차원 배열을 이용하여 보드게임에 활용하기 좋다는점
* 드래그 관련 인터페이스 들을 활용하여 쉽게 Drag & Drop 구현이 가능
* DFS방식과 백트래킹 알고리즘을 활용하여 탐색 단계를 줄일수 있다는점
